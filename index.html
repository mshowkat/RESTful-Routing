RESTful Routes

A table of all 7 RESTful routes

Name Path HTTP Verb Purpose Mongoose Method
Index /main GET List all dogs main.find()
New /main/new GET Show new main form N/A
Create /main POST Create a new main, then redirect somewhere Main.create()
Show /main/:id GET Show info about one specific main Main.findById()
Edit /main/:id/edit GET Show edit form for one main Main.findById()
Update /mains/:id PUT Update particular main, then redirect somewhere Main.findByIdAndUpdate()
Destroy /mains/:id DELETE Delete a particular main, then redirect somewhere Main.findByIdAndRemove()
